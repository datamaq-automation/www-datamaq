### Análisis y Preparación de Datos con Pandas y NumPy
Antes de entrenar un modelo, debemos poder importar y transformar los datos brutos de forma eficiente:

- **Estructuras Fundamentales**: Uso de Arrays de NumPy para cálculo numérico rápido y DataFrames de Pandas para manipulación tabular.
- **Limpieza de Datos**: Tratamiento de valores faltantes (nulls), normalización de características y codificación de variables categóricas (one-hot encoding).
- **Optimización de Memoria**: Tipos de datos óptimos para manejar sets de datos de gran escala en memoria.
