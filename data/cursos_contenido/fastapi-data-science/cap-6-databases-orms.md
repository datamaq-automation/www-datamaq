### Conectividad Asíncrona con Base de Datos
Para no bloquear el Event Loop al hacer consultas de datos, aprenderemos a configurar conexiones asíncronas:

- **SQLAlchemy Asíncrono**: Uso de `create_async_engine` y `AsyncSession`.
- **ORMs Alternativos**: Configuración de herramientas modernas como Tortoise ORM o SQLModel.
- **Migraciones con Alembic**: Manejo del ciclo de vida y la evolución de las tablas de la base de datos de forma segura.
