# Estrategia SEO - Datamaq

La migraci\u00f3n de Vue SPA a **FastAPI Jinja2 SSR** es la piedra angular de esta estrategia, eliminando las barreras de indexaci\u00f3n de las aplicaciones de una sola p\u00e1gina.

1. **Renderizado en Servidor (SSR):** Garantiza que el 100% del contenido sea visible para los rastreadores desde el primer byte.
2. **T\u00e9cnica:** Sitemap, robots.txt, cach\u00e9 HTTP.
3. **Sem\u00e1ntica:** Datos estructurados (Schema.org) y contenido alineado con AAIERIC para autoridad de marca.
4. **Presencia:** Meta Tags y Open Graph din\u00e1micos.
5. **Experiencia:** Core Web Vitals (LCP optimizado mediante SSR, sin bloqueos de renderizado JS).
6. **Autoridad:** Alineaci\u00f3n con lineamientos oficiales de AAIERIC para generar confianza.
