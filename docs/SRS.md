# Especificación de Requisitos del Sistema (SRS) - Datamaq

## 1. Funcionalidades Principales
- **Landing Page Informativa:** Muestra servicios, información de contacto y FAQ.
- **Soporte Dinámico:** Chat en vivo vía Chatwoot.
- **Cotizador (Wizard UI):** Formulario paso a paso en la web para Request for Quote (RFQ).
- **Integración RASA:** API compatible con Action Server para bot conversacional.

## 2. Requisitos de Negocio
- El sistema debe permitir la configuración de precios basados en tablas de referencia externas de AAIERIC.
- El contenido debe ser fácilmente editable mediante archivos YAML.

## 3. Estado del Proyecto
- **Cotizador (Wizard UI):** En implementación.
- **Integración RASA:** Pendiente.
